# generalizedtestmanager

A generalized Test manager

## Features

Runs all python files/tests as configured

## Extension Settings

```json
"gtm.folders_to_monitor": [
        {
            "path": "..",
            "python_path": "\\.env\\Scripts\\python.exe",
            "type": "python",
            "test_files": [
                "\\tests\\local\\test.py",
            ]
        }
    ]
```
